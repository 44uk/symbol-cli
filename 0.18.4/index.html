symbol-cli

Symbol CLI

Command Line Interface (CLI) to interact with Symbol.

:information_source: Use the software for testing/development purposes.

Important Notes
  • 0.18.1 - Fushicho5 Network Compatibility (catapult-server@0.9.3.1)

:warning: 0.18.x breaks compatibility with the profiles (public key derivation) saved using previous versions of the software. Private keys have now a new public key and address associated. Before installing symbol-cli@0.18.1, backup and delete the file ~ \.nem2rc.json.

  • 0.17.1 - Fushicho4 Network Compatibility (catapult-server@0.9.2.1)

Find the complete release notes here.

Requirements
  • Node v12.13.0 (LTS)

Installation

The Symbol CLI is distributed using the node package manager npm.

npm install -g symbol-cli

Usage

Surf the documentation to get started into Symbol development. You will find self-paced guides and useful code snippets using the Symbol CLI.

To get the full list of available commands, check the CLI reference.
